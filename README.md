## Что это

`rost/0k` — пилотный проект по практическому изучению теории самоорганизующихся систем. Граф живёт на экране сам: растёт, «ест», спит и умирает — без заданной цели и без внешнего управления. Структура возникает, а не программируется.

Открыть можно прямо в браузере (тестировалось на Safari/iPhone и десктопе) — это один HTML-файл, никакой установки.

## Что под капотом

- **Нелинейное волновое поле ψ** типа Гросса–Питаевского (уравнение с насыщением) — несущая динамика.
- **Локальное хеббовское обучение** оператора восприятия — **без глобального функционала потерь и без обратного распространения**. Все правила локальные; эмерджентность не программируется.
- **Морфогенез со свободным числом узлов** — рождение узлов по ψ-градиенту на напряжённых гранях, растворение по энергетическому бюджету.
- **Гомеостаз через энергетический бюджет** — автопоэзис: воспроизводство структуры как условие существования.
- **Орбитальный циркадный ритм** (v1.7) — солнце/луна через барицентр системы Земля–Луна, синодический месяц как биение периодов.
- **Сенсорная топография** по модальностям, миникарта.

Эмерджентная морфология доходит до самоорганизованных органов (в одном из прогонов возникла структура-«линза» при N≈155).

## Как запустить

Онлайн — включить GitHub Pages (Settings → Pages → Deploy from branch → main / root), файл `index.html` — копия для Pages.
